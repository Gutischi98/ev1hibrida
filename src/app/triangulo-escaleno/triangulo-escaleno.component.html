Triángulo Escaleno

 Calcular perímetro {{ resultado }} Perímetro del Triángulo Escaleno Como calcularlo:

En los triángulos Escalenos, cada uno de los lados tiene un tamaño distinto. Así pues, en este caso, deberás limitarte a sumar el valor de las tres longitudes para obtener así el perímetro:


P = a + b + c

Donde: a, b y c son las longitudes de los tres lados del triángulo.
